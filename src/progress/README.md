# Progress 进度条
期
用于展示操作进度，告知用户当前状态和预。
## 何时使用
在操作需要较长时间才能完成时，告知用户该操作的当前进度和状态。
+ 当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过 2 秒时； 
+ 当需要显示一个操作完成的百分比时。
